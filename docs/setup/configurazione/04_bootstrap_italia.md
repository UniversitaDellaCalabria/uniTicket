# Bootstrap Italia

Di default, il sistema si presenta con il template Bootstrap Italia.

Per modificare questo comportamento è possibile agire sul parametro *DEFAULT_BASE_TEMPLATE* specificando il file di base da far ereditare ai template del progetto.

    # default template to extend
    # Default Bootstrap Italia template: 'bootstrap-italia-base.html'
    # Example: Unical template: 'base-setup.html'
    # DEFAULT_BASE_TEMPLATE = 'base-setup.html'
