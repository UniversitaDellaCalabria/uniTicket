# Dashboard

La Dashboard del Manager è l'ambiente operativo che trasforma i dati in informazioni strategiche. Mentre la dashboard dell'utente è focalizzata sullo stato di avanzamento e quella dell'operatore sulla coda di lavoro, quella del Manager è strutturata per offrire il pieno controllo sulle risorse (gli Uffici) e sui servizi (le Tipologie di richiesta).

Si presenta come un cruscotto analitico diviso in tre aree chiave: 

* la misurazione del flusso (contatori);

* il controllo della struttura (uffici);

* la gestione dell'offerta (tipologie).

### Indicatori di Flusso e Carico Complessivo

Nella parte superiore, il Manager visualizza i badge numerici che rappresentano la salute operativa della Struttura:

* **Contatore Richieste**: riepilogo totale delle istanze divise per stato (Aperte, Assegnate, Chiuse) a livello di intera Struttura;

* **Monitoraggio Messaggi**: indicazione delle comunicazioni pendenti che richiedono un'azione, permettendo di intercettare eventuali ritardi nelle risposte agli utenti;

### Gestione degli Uffici (Le Risorse)

Il modulo dedicato agli Uffici permette al Manager di supervisionare l'organizzazione interna.  
Dalla Dashboard è possibile fare:

* **Monitoraggio Organico**: visualizzare l'elenco degli uffici attivi e il numero di operatori assegnati a ciascuno;

* **Distribuzione del Carico**: analizzare quali uffici hanno il maggior volume di richieste in gestione;

* **Accesso Rapido**: entrare direttamente nella configurazione di un ufficio per aggiungere o rimuovere personale o per modificarne le impostazioni di ricezione.

### Gestione delle Tipologie di Richiesta (I Servizi)

Questa è l'area più dinamica della Dashboard, dove il Manager controlla l'effettiva disponibilità dei servizi verso l'esterno:

* **Catalogo Attivo**: elenco completo delle tipologie di richiesta create all'interno della struttura;

* **Controllo degli Stati**: un sistema di icone segnala immediatamente se una tipologia è "Attiva", "Scaduta" o "In bozza" (non ancora pubblicata);

* **Visibilità Condizional**e: indicazione immediata di quali target (Studenti, Dipendenti, Ospiti) possono accedere a ogni specifico servizio;

* **Interruttore Rapido**: possibilità di attivare o disattivare un servizio con un click, utile per gestire emergenze o interruzioni temporanee della ricezione.
